aNOOBus
General info
It's quite simple discord bot with a lot of new features like slash-commands and discord components
aNOOBus is hosted on Heroku so there aren't any issues with connections. There can be issues with databses cause I use free cloud database with 100MB limit so it can just disappear in any time)
Bot itself
There are a lot of features in the bot(not), it has 4 categories in it:
  • Moderation(no explanation needed)
  • Fun(Commands like punch, kill, hug)
  • Slash-commands(In that category there are all commands that r comfortable to use like slahs-commands)
  • Filters(Command that can upgrade image)
    • Now about commands
    Here I will make an examples with the best commands from each category, u can view all of them by typing "!!help"(u need to send message) or "/"(and just view bot slash-commands), so here we go:
    Moderation:
    !!createjoins {category name} - creates voice channel that will create private voice channels(like voicemaster), but aNOOBus will give channel owner permissions to manage channel as he want, voicemaster doesn't have that feature
    !!setlogs {channel name} - will set server logs to mentioned channel
    Fun:
    !!action {mention} - bot will send list of actions that u can do with member(like hug, kiss, etc)
    /Slahs-Commands
    /gradient {rgb code of color 1} {rgb code of color 2} opttional:{rgb code of color 3} - will generate gradient from given colors
    /chemist {1st element} {2nd element} - will solve chemistry equation(sum of 2 elements)
    Filters:
    !!filter {image} - bot will ask what to do with image and implement chosen filter on image
